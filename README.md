# IsmaCut

基于 Double-Array Trie 的中文分词工具。使用 DAG（有向无环图）+ 动态规划求解最大概率路径，实现高效的中文分词。

## 原理

1. **Double-Array Trie** (`trie.h`) — 将词典构建为双数组 Trie，支持精确查找和前缀搜索
2. **DAG 构建** — 对输入句子的每个位置，通过前缀搜索找出所有可能的词，构建有向无环图
3. **动态规划** — 基于词频取对数作为权重，从右向左计算最大概率路径
4. **分词输出** — 沿最优路径切分句子；未登录字按单个 UTF-8 字符回退

## 构建

```bash
make
```

## 使用

### 交互模式

准备一个词典文件，每行格式为 `词\t词频`，例如：

```
南京	100
南京市	80
市长	50
长江大桥	120
```

启动交互式分词：

```bash
./cut dict.txt
```

```
> 南京市长江大桥
南京市/长江大桥
```

### 运行测试

不带参数运行即可执行内置测试：

```bash
./cut
```

## 项目结构

```
trie.h   - Double-Array Trie 实现（构建、精确查找、前缀搜索）
cut.h    - 分词器声明及词频权重计算
cut.cc   - DAG 构建、动态规划求解、分词主逻辑
main.cc  - 词典加载、交互式 REPL、内置测试
```
